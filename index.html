Las directivas son la herramienta que ofrece Vue para poder manipular el DOM y para poder hacerlo de forma declarativa.

Directiva Significado Sintaxis Resultado
v-text
 Actualiza el textContent del elemento. v-text="msg"
v-html
 Actualiza el innerHTML del elemento. Note que los contenidos son insertados como HTML puro - no serán compilados como pantillas (templates) de Vue v-html="html"
v-bind
 Enlaza dinámicamente atributos de un elemento. v-bind:src="url" o :alt="name"
Usando v-bind para cambio de clases con comparaciones v-bind:class="changePercent > 0 ? 'green' : 'red'"

Clases en tiempo real

Usando v-bind para cambio de clases con Objetos v-bind:class="{orange: changePercent === price, red: changePercent < price, green: changePercent> price}"

Clases en tiempo real
Usando v-bind para cambio de estilos v-bind:style="{background:color}"

Ingerese un Hexadecimal para cambiar el color de fondo.
v-if
 Construye o destruye un elemento de acuerdo a la condición dada. v-if="changePercent > 0" :D :D :|
v-else-if
 Equivalente a “v-if”, se usa despues de un “v-if” o “v-else-if”. v-else-if="changePercent < 0"
v-else
 Se activa cuando las condiciónes "v-if" o "v-else-if" no se cumplen v-else
v-show
 Cambia la propiedad CSS display del elemento basado en las condiciones dadas. v-show="changePercent > 0" :D
v-model
 Crea un enlace bidireccional (two way binding) en un elemento de formulario o componente. v-model="inputText"

'InputText' es igual a: {{inputText}}
v-on
 Agrega una escucha de evento (event listener) al elemento. v-on:click="toggleHidePrices" o @click="toggleHidePrices"
v-on:mouseover="toggleHidePrices" @mouseout="toggleHidePrices"
v-for
 Renderiza el elemento o plantilla mutiples veces basado en la fuente de información. v-for="item in items"
